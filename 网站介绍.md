# 🏮 天灯 · 许愿星河

> 一个温暖治愈的虚拟天灯放飞平台 —— 写下心愿，点燃天灯，让它飘向璀璨星河。

---

## 🌌 这是什么？

**「天灯 · 许愿星河」** 是一个在线虚拟孔明灯放飞网站。

在深邃的星空下，你可以写下藏在心底的愿望，选择一盏属于你的天灯颜色，亲手将它点燃，看着它带着你的心愿缓缓升空，飘入满天星辰之中。与此同时，你也能看到夜空中来自陌生人的千万盏天灯——点击它们，就能读到别人的故事。

**它是一个许愿池，一个树洞，也是一片由无数陌生人的善意点亮的星河。**

---

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🔥 **放飞天灯** | 写下心愿 → 选择颜色 → 点燃放飞，天灯从底部弹入天空 |
| 🎆 **粒子特效** | 放飞瞬间火花四溅 + 屏幕微震，仪式感拉满 |
| ✨ **心愿文字** | 所有心愿化为发光文字，在星空背景中缓缓飘浮 |
| 👆 **阅读心愿** | 点击夜空中任意天灯，阅读陌生人的愿望 |
| ❤️ **温暖点赞** | 给被打动的心愿送上一份温暖 |
| 🌠 **动态星空** | Canvas 实时渲染的闪烁星空 + 随机流星 |
| 📱 **全端适配** | 手机 / 平板 / 电脑都能完美体验 |
| 💾 **本地存储** | 心愿保存在浏览器中，下次回来它们还在 |

---

## 🎨 视觉亮点

- **深邃夜空**：从深蓝色渐变到近乎黑色的星空画布，数百颗闪烁的星星
- **发光天灯**：六种颜色可选（暖橙/正红/金色/粉色/紫色/青色），每盏都带有温暖的光晕
- **粒子爆发**：天灯放飞时火花从屏幕底部炸开，配合屏幕微震
- **心愿星河**：所有心愿以发光文字的形式在背景中缓缓飘浮
- **山峦剪影**：底部远山和水面倒影，营造沉浸式的夜晚氛围
- **随机流星**：偶尔划过的流星，给星空增添惊喜

## 🛠 技术栈

纯前端实现，无需服务器：

- **HTML5 Canvas**：星空渲染 + 粒子系统
- **CSS3**：天灯光晕、浮动文字、响应式布局
- **JavaScript (Vanilla)**：天灯物理模拟、localStorage 数据持久化
- **GitHub Pages**：免费托管部署
