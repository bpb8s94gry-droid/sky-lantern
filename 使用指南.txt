# 🏮 天灯·许愿星河 - 使用指南

## 这是什么？

一个温暖的虚拟天灯放飞网站。人们写下心愿，点燃天灯，看着它飘向星空。
其他人可以看到满天漂浮的天灯，点击就能读到陌生人的心愿，还能点赞送温暖。

## 怎么玩？

1. 点击「放飞我的天灯」或按空格键
2. 写下你的心愿（最多140字）
3. 选择天灯颜色
4. 点击「点燃·放飞」
5. 在天空中点击任意天灯，阅读别人的心愿
6. 给你喜欢的心愿点个赞

## 怎么靠这个赚钱？不圈钱的方式

- 网站流量大了之后，在页面底部加一个低调的赞赏码（微信/支付宝收款码）
- 卖出「限定颜色天灯」——比如金色天灯 1元
- 连接广告平台（Google AdSense 等），靠流量赚广告费
- 在社媒（小红书/抖音/B站）发视频展示这个网站，引流

## 怎么让网站在网上被搜到？

### 免费方式：GitHub Pages
1. 注册 GitHub → 新建仓库 → 上传 index.html
2. 开启 GitHub Pages → 获得免费网址

### 买域名更专业
1. 阿里云/腾讯云买域名，约 30-60元/年
2. 用 Vercel/Netlify免费绑定域名

## 修改指南

你可以搜索修改这些：
- 搜索「demoWishes」→ 改预设心愿
- 搜索「title」→ 改网站标题
- 搜索「140」→ 改心愿字数上限
- 搜索「0a0a1a」→ 改夜空背景色
